Project Description

Mardal, a marketing company, commissioned the design and development of a new website with a strong focus on human-centered experience, clarity, and technical robustness.

The project covered the full process from UX and UI design to implementation in WordPress, combining interaction design, visual hierarchy, and structured user flows with a maintainable front-end architecture. Design decisions were guided by human-centered and cognitive psychology principles, informed by my academic background in psychology, to reduce cognitive load and support intuitive navigation. Development was executed using HTML, CSS, and JavaScript, enabling custom interactions while maintaining performance and responsiveness.

A key element of the experience is the use of smooth, scroll-triggered animations implemented as functional interaction cues, progressively revealing content to guide attention and reinforce hierarchy without compromising usability or accessibility.

Less, but intentional.

This project follows a single rule: every element must earn its place.

Space, typography, and motion are used as functional tools—not decoration.
