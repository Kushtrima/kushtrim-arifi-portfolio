PROJECT

This project was commissioned by MONUN AG, for their brand Stplanner, with the goal of creating a modern and minimal corporate website that reflects professionalism, clarity, and trust.


I started with UX research and Figma prototyping, exploring layouts that balance visual simplicity with information hierarchy. After presenting several design directions, the final concept was approved by MONUN AG, and I developed it from scratch using HTML, CSS, and JavaScript.


The website features a glass-style header that transitions smoothly on scroll, subtle animations, and a responsive grid structure optimized for all screen sizes. Each visual and functional choice was made to communicate transparency and precision—values central to the Stplanner brand.

MY ROLE

UX/UI Design: Research, wireframing, and final layout in Figma

Front-End Development: Hand-coded structure, style, and animations

Optimization: Performance tuning and responsive testing

TECH STACK

Figma · HTML · CSS · JavaScript · WordPress (custom integration)

SUMMARY

The main goal of this build was to keep the website as efficient and lightweight as possible without sacrificing aesthetics or functionality. I chose to hand-code everything with HTML, CSS, and JavaScript instead of using frameworks, to ensure faster load times, full control over animations, and a clean codebase that can easily be maintained or updated in the future.


Using custom CSS and minimal scripts allowed me to fine-tune every transition and effect, achieving a smooth, consistent behavior across all browsers. This approach also reduced unnecessary dependencies and improved long-term stability.


From a design perspective, the focus was on visual balance and usability — keeping a clear hierarchy, readable typography, and simple motion. The goal wasn't to decorate, but to make each interaction feel natural and aligned with the brand's precise and transparent identity.
