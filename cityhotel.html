Project Description

Cityhotel is a four-star hotel located in Näfels, Switzerland, and includes a dedicated restaurant named Glaré. The project was requested by Monun AG, and my role was to complete the website as a full project based on the provided requirements.

After receiving the initial ideas and content, I conducted research on the Swiss hotel website market. During this process, I identified functionality as a primary focus across similar websites, which directly informed my overall approach to the project.

I designed the UX/UI with an emphasis on clarity and easy navigation, integrating all required website functions. The structure includes a minimal header, a large menu, multiple reservation options, and a secondary page behavior where the header logo changes when navigating to the restaurant section.

Following the design phase in Figma, the website was developed using WordPress, with JavaScript used for specific functionality and CSS applied for global styling. The final website uses smooth interactions and a friendly color palette.

Elegant simplicity, refined experience.

Every element is designed to enhance usability while maintaining a premium aesthetic.

Clean layouts and thoughtful interactions guide guests through the booking journey seamlessly.
